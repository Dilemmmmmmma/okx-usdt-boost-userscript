# OKX USDT Boost Userscript

Tampermonkey install URL:

https://raw.githubusercontent.com/Dilemmmmmmma/okx-usdt-boost-userscript/main/OKX-USDT-Boost.user.js

## Auto Update

Tampermonkey checks this file for updates:

https://raw.githubusercontent.com/Dilemmmmmmma/okx-usdt-boost-userscript/main/OKX-USDT-Boost.user.js

When editing the script, update both places:

- `OKX Web3 DEX USDT 盈亏计算器 (含 Boost 计算、自动刷新与跨域通讯) 极简版-6.15_Clean_UI.js`
- `OKX-USDT-Boost.user.js`

Every published change must bump the userscript `@version`, otherwise Tampermonkey may not detect the update.

## Publish

Create this GitHub repository first:

https://github.com/new?name=okx-usdt-boost-userscript&visibility=public

Then push from this folder:

```powershell
git push -u origin main
```
