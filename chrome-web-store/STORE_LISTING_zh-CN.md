# Chrome Web Store 商店资料

## 名称

Boost & Alpha 交易助手

## 简短说明

在 OKX Web3 与 Binance Alpha 页面中提供本地交易统计、参数管理和用户启动的自动交易辅助。

## 单一用途

为受支持的 OKX Web3 Boost 与 Binance Alpha 交易页面提供统一的本地交易辅助工作台。

## 详细说明

Boost & Alpha 交易助手在 Chrome 侧边栏中提供统一的交易工作台，可读取当前受支持页面的代币、订单和活动状态，保存本地参数，并在用户明确启动后执行预设交易流程。

主要功能：

- OKX Web3 Boost 交易统计与进度展示；
- Binance CEX Alpha 交易辅助；
- Binance Wallet Alpha 买入、卖出与订单确认；
- 本地保存交易参数、状态与统计记录；
- 页面切换、订单延迟和余额尚未到账时的状态恢复。

重要说明：

- 本扩展是第三方工具，并非 OKX、Binance 或 Google 官方产品；
- 本扩展不会读取或保存私钥、助记词、密码、MFA 密钥或一次性验证码；
- 所有交易均通过用户当前已登录的第三方页面执行；
- 自动化交易存在滑点、手续费、行情波动、网络延迟与资产损失风险；
- 用户应先使用小额资金验证参数，并自行承担交易决定与结果。

## 建议分类

生产力工具

## 权限说明

### sidePanel

用于在浏览器侧边栏中展示交易工作台和运行状态。

### scripting

用于把扩展包内的本地交易引擎注入当前受支持页面。扩展不下载或执行远程代码。

### 站点访问权限

仅用于在 OKX Web3 与 Binance 的受支持页面中读取代币、订单、余额和交易状态，并执行用户明确启动的交易操作。

## Privacy practices 建议披露

- 网站内容：是，本地读取受支持交易页面中的订单、余额、代币和活动状态；
- 金融与付款信息：是，本地处理交易金额、余额和订单结果；
- Web 浏览活动：是，仅判断当前标签是否属于受支持页面；
- 个人身份信息：建议勾选，因为钱包地址或平台账户标识可能构成用户标识；
- 身份验证信息：否，扩展不读取密码、Cookie、MFA 密钥或验证码；
- 数据出售或用于广告：否；
- 远程代码：否；
- 数据传输给开发者服务器：否。

## 审核备注

本扩展只在清单声明的域名中运行。所有页面引擎均随扩展包提供，不使用远程脚本。扩展不会自动启动交易；交易流程由用户在侧边栏中明确点击启动。设置和统计数据保存在用户设备本地。
