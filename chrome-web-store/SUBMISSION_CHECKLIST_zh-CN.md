# Chrome Web Store 提交清单

- [ ] Chrome Web Store 开发者账号已注册并启用两步验证
- [ ] 上传发布目录中的 ZIP 文件
- [ ] 主语言选择简体中文
- [ ] 填写商店名称、简短说明、详细说明和单一用途
- [ ] 上传 128×128 商店图标
- [ ] 上传至少一张 1280×800 PNG 或 JPEG 截图
- [ ] 填写 `sidePanel`、`scripting` 与各站点权限用途
- [ ] Remote code 选择“否”
- [ ] 按商店文案披露本地处理的网站内容、金融信息、浏览活动和用户标识
- [ ] 填写公开可访问的隐私政策 URL
- [ ] 选择公开、非公开或可信测试者分发范围
- [ ] 首次建议关闭自动发布，审核通过后再手动发布

注意：公开扩展涉及加密资产自动交易，可能需要更长的人工审核时间。商店资料不得暗示本扩展由 OKX、Binance 或 Google 官方开发或背书。
